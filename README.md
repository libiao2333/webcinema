# webcinema
WebCinema 是一个基于 Flask 的高性能本地媒体服务器，专为浏览和播放本地存储的图片、视频、音频和文本文件而设计。它提供了现代化的 Web 界面，支持硬件加速解码、智能格式转码和便捷的文件管理功能。

------------

### 如何安装

1、下载install.exe

2、双击运行并根据指引即可
